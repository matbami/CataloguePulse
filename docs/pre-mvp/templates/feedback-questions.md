# Feedback And Pricing Questions

Ask these after sending the audit report.

## Core Feedback

1. Was this audit useful?
2. Which issue felt most important?
3. Was anything in the report surprising?
4. Do you currently check these issues manually?
5. Who usually handles product page cleanup for your store?
6. How many products do you have?
7. How often do you add new products?

## Product Interest

1. Would you use a Shopify app that scans your full catalog for these issues?
2. Would you prefer a weekly scan, monthly scan, or one-time cleanup?
3. Would you want suggested fixes to apply automatically or only after approval?
4. Which feature would matter most first?
5. What would stop you from installing it?
6. Would you join an early beta?
7. Can I follow up when the first version is ready?

## Pricing Validation

Avoid asking only "would you pay?"

Ask:

1. What would make this worth $29/month?
2. Would $19-$49/month feel reasonable if it scanned your catalog and helped fix issues in bulk?
3. Would you prefer monthly subscription or pay-per-cleanup?
4. How many products would you need to fix each month?
5. Would you pay more for bulk updates after approval?
6. Would this be more valuable if it helped with Google Shopping feed readiness?
